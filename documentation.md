# Документация ARGON.ru

## Как установить

1. Скачайте репозиторий через `git clone https://github.com/Wert1209yt/argon-server` или через скачать ZIP на главной странице нажав на кнопку code
2. Установите PHP 8.x, инструкция по его установке есть в интернете.
3. Зайдите в папку проекта и напишите команду `php -S 0.0.0.0:8000 router.php`
4. Сервер успешно запущен.
5. Опционально вы можете изменить настройки в config.php (в папке src)

## Патч клиента

Проект изначально создавался под приложение OZON.ru версии 1.5.3

1. Найдите APK этой версии.
2. Декомпилируйте его с помощью ApkTool
3. Найдите в папке приложения AndroidManifest.xml и найдите тег application и добавьте внутрь него `android:usesCleartextTraffic="true"`, сохраните файл.
4. Зайдите в папки `smali/ru/ozon/app/android` и замените в файлах `Constants.smali` и в папке `Helpers` `UrlBuilder.smali`, `ApiHelper.smali` https://ows.ozon.ru на ваш IP (например http://192.168.3.6, без буквы s!).
5. Скомпилируйте и подпишите (можно и testKey) приложение.
6. Готово! Если вы уже запустили сервер, то уже можете запускать!
